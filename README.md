# ⬇️ Instalação

Primeiro de tudo, para poder instalar o script precisará do [termux](https://google.com) atualizado
# Comandos iniciais:
```
apt update
```
```
apt upgrade
```
```
termux-setup-storage
```
```
pkg install git
```
```
pkg install python
```
# Agora sim instale o script:
```
git clone https://github.com/venomkkk/wa
cd wa
python install.py
python main.py
```
